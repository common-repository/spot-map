 == Installation ==
1. Unpack the *.zip file and extract the /spotmap/ folder.
2. Drop the ‘spotmap’ folder into your ‘wp-content/plugins’ folder
3. In word press administration panels, click on plug-in from the menu.
4. You should see your new ‘Online People Map ’ plug-in listed under Inactive plug-in tab.
5. To turn the word presses plug-in on, click activate.
6. Just copy and paste this code <!--spotmap form--> in your page or post where you want your map to appear

Parameters

Parameters

Your ApiCode: you can get the free api code from this site: http://code.google.com/apis/maps/signup.html

Width and Height: you can mention your map width and height without px.

for example 200px=200 without px.

Latitude and Longitude : you can get the latitude,longitude from this site

http://universimmedia.pagesperso-orange.fr/geo/loc.htm

Title: Your preffered title

Small description: Description about the place.

Link: you can give your website url

Text to link: the text name which you wan to link

for example  you want your name to be a link then give myname

== Frequently Asked Questions ==

I installed the plugin but can’t see it. What’s wrong?
Make sure the directory name is entered correctly on the settings page.

Map title appears but no map appear below it?
Please check your api,key, and all the latitude,longitude parameters, it should not be blank .

Map appears but its too short?

Please cheack your width and height .
